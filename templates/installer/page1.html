Solar Imperium 3.0 Installer (Page 1 of 4) :: Introduction

Welcome to the Solar Imperium 3.0 installer. This project is a community-developed recreation of the classic 1990s strategy game, updated to run on modern systems.

To continue, please ensure your server meets the following basic requirements:

  • A web server (Apache recommended)
  • PHP 8.5 or later
  • A database server (MariaDB recommended)

Once these requirements are met, click Continue to proceed with installation.

Note: Solar Imperium supports multiple languages, but the installer itself is provided in English.
